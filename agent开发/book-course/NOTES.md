# NOTES — 《深入理解 AI Agent》精读课工作区

## 这门课是什么
以李博杰《深入理解 AI Agent：设计原理与工程实践》(10 章 + 引言 + 后记) 为真实源，生成的一套 **57 节 HTML 微课**。原始资料在 `../原始资料/ai-agent-book/`（`book/*.md` 是正文，`chapterN/` 是配套开源实验代码）。

## 与本工作区另两门课的关系（互补，别混）
- `from-zero/`（入门·发育学）与 `lessons/`（进阶·四支柱解剖学）——都以用户自己的生产项目 **`mino_server`** 代码为真实源，句句落到文件名+行号。
- `book-course/`（本课·书精读）——以**书**为真实源，句句落到书里的**概念、图号、实验编号、开源项目目录**。**不写 mino_server 的代码行号**，那是另两门课的风格。

## 房屋风格（drafter 必守）
- 复用共享样式：`../../assets/style.css` + `../../assets/diagram.css`；本课专属组件在 `assets/book.css`（`.formula` / `.pitfall` / `.experiment` / `.bookquote` / `.trio` / `.lesson-nav`）。
- 视觉/结构的**唯一基准是 `lessons/0001-agent-core-formula.html`**（已浏览器验收）。新课一律照它的骨架浇。
- 每课六件套：导语(lede) → 2–5 个 `<h2>`（是什么→为什么→边界）→ ≥2 幅图(≥1 信息图) → `.pitfall` 常见误区 → `.keydef` 关键定义 → `.quiz` 自测 → `.good` 一句结论 → 主推资源 → footer 导航。
- 纯 CSS/HTML/内联 SVG，**零外链**（本地 `open` 打开就得能看，深色模式必须可读，SVG 用 `dgm-*` class）。

## 防幻觉纪律（最重要）
- 只写原文能支撑的话；不确定就不写。图号只用书里真实出现的（如「图4-3」），项目目录只用 `chapterN/` 里真实存在的。
- 术语用书的口径（见 `GLOSSARY.md`，86 条，全部取自原文定义）。
- 「唯一/全部/必然」等绝对措辞是幻觉指纹，落笔前先在原文找反例。

## 教学法（授课时）
- 循序渐进，一次一个概念；进入下一课前先让学习者复述，再补缺口。
- 用 AskUserQuestion 出选择题校验理解，正确项打乱位置、选项等长。
- 鼓励动手：讲到有配套实验的章节，推学习者去跑 `chapterN/` 对应项目。

## 构建溯源
本课由三步生成：① 11 个 subagent 并行精读全书 → 结构化 course-map（57 候选微课 / 86 术语 / 48 配套项目）；② 手写并浏览器验收金标准 0001；③ 56 个 subagent 各据其 spec + 章节原文克隆 0001 骨架产出 0002–0057。course-map 原始数据见会话记录。
