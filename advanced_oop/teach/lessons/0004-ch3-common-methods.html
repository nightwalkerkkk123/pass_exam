主题简介

第 3 章聚焦"覆盖 Object 通用方法时的常见陷阱"——主要就是 equals、hashCode、toString 这三个。所有 Java 类默认继承自 Object，所以这些方法的覆盖质量直接影响所有使用 HashMap / HashSet / 日志 / 调试的场景。

本节 3 条都是高频考点：

  • 条款 10：equals 覆盖必须满足 5 性质（自反 / 对称 / 传递 / 一致 / 非 null），破任一 → 集合行为诡异
  • 条款 11：equals 相等 ⇒ hashCode 必须相等；反之不要求——HashMap/HashSet 靠这条
  • 条款 12：toString 一定要写好——默认 PhoneNumber@163b91 难调试
