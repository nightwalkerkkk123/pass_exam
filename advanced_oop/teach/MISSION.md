# Mission: Advanced OOP 期末复习（过线型）

> **变更记录**：v2（2026-06-13）。v1 假设用户有基础且想"应试+真会用"；诊断课（[[learning-records/0001]]）显示用户零词汇基础，遂改为"过线型"。

## Why
一周内通过 `advanced_oop` 期末考试。**过线**是唯一目标，不追求深度掌握，不追求"真会用"——这两件事在 1 周零基础的时间窗下互斥。

## Success looks like
- 闭卷时能**复述**第 2–8 章约 20 条高频条款的**名称**和**核心反例**（反例 = 违反了这条会写出什么烂代码）
- 看到一段 Java 代码，能用"条款 N"指认其中违反/遵守的条款
- 在选择题 / 简答题题型上命中率 ≥ 60%（过线型档位，低于 v1 的 85%）
- 完整体验过 1 份过线级条款速查表（`reference/cheatsheet.html`）

## Constraints
- **时间：1 周内**——单 lesson 5–10 分钟
- **基础：零词汇**——所有 lesson 必须从名词解释开始，不能跳过术语
- **深度档位：过线**——每章只挑 2–3 条高频；低频 / 高级用法主动放弃
- **资料源**：以 `advanced_oop/` 既有的 4 份 Markdown 复习资料为权威源
- **不联网**——无可信外网资源时，标 Gaps，不杜撰

## Out of scope
- "真会用"层面的代码练习、应用场景
- 低频条款与高级用法（枚举策略模式、Stream 复杂编排等）
- Effective Java 第 1 章 / 第 9 章及之后
- Java 语法基础、设计模式系统学习、JVM 内部机制
- 长线掌握

## Schedule 思路（不锁死）
- 第 1 天：词汇地基（Lesson 0002）+ 速查表（`reference/cheatsheet.html`）
- 第 2 天起：按高频条款分章短课，每章 ≤15 分钟
- 每天 1 个 learning-record 收口
- 第 6–7 天：模拟自测 + 错题回炉
