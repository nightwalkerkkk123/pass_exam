# Advanced OOP 复习 Glossary

本工作区使用的术语表。Effective Java 条款相关的术语**不在 user 真正理解前进入本表**——定义本身就是 user 学会的证据。

## 工作区元术语

**Lesson**:
本工作区的最小教学单元，一个独立的 HTML 文件，教授一件与 mission 直接相关的事。
_Avoid_: 课件、教程、章节

**Learning Record (`./learning-records/0001-...md`)**:
教学场景的"架构决策记录（ADR）"——记下 user 真正理解（或被纠正）的非显然知识点。**不是**日志，**不是**错题本。
_Avoid_: 错题集、笔记、上课记录

**ZPD (Zone of Proximal Development)**:
最近发展区——user 当前能力与"再踮一下脚就能到"之间的差距。Lesson 必须落在这个区间内，否则太简单无聊或太难放弃。
_Avoid_: 学习范围、难度等级

**Mission**:
写在 `MISSION.md` 里的"为什么学"——所有 lesson 选材、RESOURCES 优先级、深度档位的指南针。
_Avoid_: 学习目标、计划

**条款 N**:
Effective Java 第 N 条建议（Item N）。本工作区统一用此引用格式，与 `Effective_Java_期末复习资料.md` 对齐。
_Avoid_: 知识点、规则
