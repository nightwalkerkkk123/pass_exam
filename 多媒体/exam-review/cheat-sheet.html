流程 / 步骤

JPEG 编码 5 步

1. RGB → YCbCr + 4:2:0 子采样(可逆,视觉无损)
2. 分块 8×8 + 2D DCT(无损可逆,能量集中)
3. 均匀标量量化 F̂ = round(F/Q)  ← 不可逆,JPEG 有损主因
4. ZigZag 扫描 + DC DPCM / AC RLC(无损)
5. Huffman 熵编码(无损,去统计冗余)

H.261 P 帧编码

宏块 → ME 找 MV → MC 取预测 → 残差 DCT → 量化 → ZigZag → Huffman
参考帧必须用 已解码的帧(防 drift,DPCM 类系统共性)

03.主题卡 3 · 质量度量与编码

覆盖 L05 / L06 / L07 / L11 · SNR/PSNR/SQNR/熵/Huffman

关键公式

名称公式说明
SNR(信噪比)SNR = 10·log10(Ps / Pn)Ps=信号功率,Pn=噪声功率
PSNRPSNR = 10·log10(x²peak / MSE)8-bit 灰度 x_peak=255
SQNR(量化信噪比)SQNR = 6.02 × N (dB)N=量化 bit 数
熵η = −Σ pi·log2(pi)信息下界
Huffman 界η ≤ l̄ < η+1整数约束 + Kraft
MSEMSE = (1/N)·Σ(xi − x̂i)²PSNR 的分母
Nyquist 定理f_sample ≥ 2·f_max采样率至少 2 倍信号最高频

关键数字 / 阈值

项目数值
8-bit SQNR48 dB(电话语音)
16-bit SQNR96 dB(CD 音质)
CD 采样率44.1 kHz(对应 22.05 kHz 信号)
电话采样率8 kHz(对应 4 kHz 信号)
电话码率8 kHz × 8 bit = 64 kbps(DS0)
每 +1 bit+6 dB SQNR
MSE=100 时 PSNR≈ 28 dB

易混对比

对比关键差异
无损 vs 有损无损:Huffman/LZW/算术,l̄ ≥ η;有损:JPEG/MPEG,PSNR-CR 曲线
Huffman vs 算术编码Huffman:整数码字,接近熵;算术:分数码字,更接近 η,但实现复杂
SNR vs PSNRSNR:功率比,一般信号;PSNR:用峰值归一化,图像
MSE vs MADMSE:平方误差,PSNR 用;MAD:绝对误差,运动估计用
DPCM vs ADPCMDPCM:固定量化步长;ADPCM:步长自适应
DM vs DPCMDM = DPCM 的 1-bit 简化(只输出方向)

口诀 · 速记

  • "功率 10,电压 20" — dB 公式永远 10·log10(功率比);电压/电流用 20·log10
  • "SQNR ≈ 6N dB" — N 位量化 → 大约 6N dB 信噪比
  • "Huffman l̄ ≥ η" — 平均码长必 ≥ 熵,只有所有 -log2(pi) 为整数时 l̄ = η
  • "Nyquist 2 倍" — 采样率 ≥ 2 倍信号最高频率

口诀 · 速记

  • "SIFT 不抗仿射、MSER 抗仿射" — 唯一抗仿射的检测子
  • "Harris 单尺度" — Harris 不抗缩放,Harris-Laplace 才抗
  • "ResNet 解决深度退化" — 不是梯度消失,而是训练误差随深度↑
  • "池化零参数" — MaxPool/AvgPool 无学习权重
  • "Dropout 测试关闭" — inverted dropout 训练时已除以 (1-p)
  • "DoG 简化 LoG" — SIFT 用 DoG 近似 LoG,计算更简单
