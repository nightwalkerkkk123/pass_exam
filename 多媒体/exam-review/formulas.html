公式手册 · 考前 1 小时过一遍

22 个核心公式按主题聚类 · 11 个讲次全覆盖 · 数字即记忆的钩子

本手册收录期末最易出计算题和应用题的 22 个核心公式,按"图像 → 颜色 → 视频 → 音频 → 压缩 → 检索 → 深度学习"7 大主题聚类。每条公式注明讲次来源和典型陷阱,建议考前 1 小时过一遍。

1 · 图像数据(L02)

灰度公式
Y = 0.299·R + 0.587·G + 0.114·B(BT.601)
数据量
Size = W × H × B / 8 (Bytes);B = 每像素 bit 数
RGB → CMY
C = 1 − R, M = 1 − G, Y = 1 − B(归一化 [0,1])
Bayer 抖动
O(x,y) = 1 if I(x,y) > D(x mod n, y mod n) else 0

陷阱:灰度公式 G 通道权重 0.587 最大(人眼对绿最敏感);数据量单位换算 1024 而非 1000。

2 · 颜色空间(L03)

Gamma 显示
L_display = V^γ;γ ≈ 2.2(CRT/sRGB);inverse gamma 预校正
YCbCr 4:2:0
每 2×2 像素块 = Y(4)+Cb(1)+Cr(1) = 6 字节 → 1.5 B/px
YCbCr 4:2:2
水平减半 → 2.0 B/px
YCbCr 4:4:4
无子采样 → 3.0 B/px(等同于 RGB 24-bit)
RGB → Y
Y = 0.299R + 0.587G + 0.114B;Cb/Cr = 128 + 色差

陷阱:YCbCr ≠ YUV(YCbCr 是 YUV 经 scale+offset 的数字版);Y 范围 16-235,Cb/Cr 范围 16-240。

3 · 视频基础(L04)

视频数据量
Size = W × H × fps × 时长 × 每像素字节
Nyquist
f_sample ≥ 2·f_max;假频 = |f_sample − f_true|
扫描帧率
NTSC 29.97 fps 隔行;PAL 25 fps 隔行;HDTV 1920×1080 50/60p
码率
码率 = 像素数 × fps × BPP(BPP = bits per pixel)

陷阱:1080p@30fps 4:2:0 = 1920×1080×30×1.5 ≈ 93 MB/s(原码率约 744 Mbps)。

4 · 音频基础(L05)

SNR(信噪比)
SNR = 10·log10(Ps/Pn) = 20·log10(Vs/Vn)(功率 10,电压 20)
SQNR(量化信噪比)
SQNR = 6.02·N (dB);N = 量化位数;每 +1 bit ≈ +6 dB
码率(码率 = 数据率)
码率 = 采样率 × 量化 bit × 通道数(单位 bps)
DPCM 差分
d[n] = x[n] − x[n−1];残差编码
电话 PCM
8 kHz × 8 bit = 64 kbps(DS0 标准)

陷阱:SNR 系数 10 vs 20 是功率/电压之分,别记混;16-bit CD ≈ 96 dB SQNR。

5 · 无损压缩(L06)

自信息
I(s) = −log2(p(s));单位 bit
信息熵
η = −Σ pi·log2(pi);熵 = 平均自信息
平均码长
l̄ = Σ pi·li;li = 第 i 符号码长
Huffman 界
η ≤ l̄ < η+1;整数约束 + Kraft 不等式
压缩比
CR = B0 / B1;B0 原始,B1 压缩后;CR>1 即压缩
Kraft 不等式
Σ 2^(−li) ≤ 1;前缀码存在性

陷阱:Huffman 是 l̄ ≥ η(不是等于);算术编码可以无限逼近 η(理论极限)。

6 · 有损压缩(L07)

MSE(均方误差)
MSE = (1/N)·Σ(xi − yi)²;xi 原始,yi 解码
PSNR(峰值信噪比)
PSNR = 10·log10(x²peak / MSE);8-bit xpeak = 255
量化
F̂(u,v) = round(F(u,v) / Q(u,v));Q 是量化步长
DCT(1D)
F(u) = C(u)·Σf(i)·cos[(2i+1)uπ / (2N)];C(0)=√(1/N),C(u)=√(2/N)
率失真
R(D) 曲线下凸;D 越大 R 越小;D=0 时 R=η(无损)

陷阱:DCT 本身无损,有损来自量化;PSNR 永远是 10·log10(功率意义)。

7 · JPEG 量化因子(L08)

条件公式说明
qf ≥ 50scaling = (100 − qf) / 50qf=50 → scaling=1(标准表);qf=75 → 0.5(更精细)
qf < 50scaling = 50 / qfqf=25 → scaling=2(粗糙);qf=10 → scaling=5

陷阱:qf 越高 scaling 越小,量化越精细,文件越大;qf 越低 scaling 越大,量化越粗糙,文件越小但失真大。

8 · 视频压缩(L09)

MAD(平均绝对差)
MAD = (1/N²)·Σ|T(i,j) − R(i,j)|;MV 搜索目标
DPCM(DC 系数)
d_i = DC_{i+1} − DC_i;相邻块 DC 差分
搜索窗口
(2p + 1) × (2p + 1);p = 搜索半径
p = 15 实例
31 × 31 = 961 个候选位置;复杂度 O(N²)
H.261 宏块
16×16 Y + 8×8 Cb + 8×8 Cr(4:2:0 下)

陷阱:MAD 不是 MSE,前者用绝对值,后者用平方;搜索窗口 = (2p+1)²。

9 · 帧类型压缩比(L10)

帧类型压缩比说明
I 帧≈ 7:1帧内压缩,中等压缩率,作为参考帧
P 帧≈ 20:1前向预测,利用时间冗余
B 帧≈ 50:1双向预测,最高压缩率
平均≈ 27:1MPEG 标准压缩水平

陷阱:I : P : B = 7 : 20 : 50,平均 ≈ 27:1;B 帧需前后两个参考帧,延迟高。

10 · CNN 输出尺寸(L12)

输出尺寸
(N + 2·padding − F) / stride + 1;N 输入边长,F 卷积核
参数量
(F·F·C + 1)·K;F 卷积核,C 输入通道,K 卷积核数
池化输出
(N − F) / stride + 1;无填充(常见 max pooling)
典型实例
输入 224×224, F=3, stride=1, padding=1 → 224×224(保持尺寸)
AlexNet 首层
F=11, stride=4, padding=2, N=224 → 55×55×96(96 个卷积核)

陷阱:公式里 2·padding 表示左右(或上下)各填 padding,常被漏算一边。

11 · 检索与特征(L11)

Harris 角点
R = det(M) − k·trace(M)²;k ≈ 0.04-0.06;R > 阈值即角点
SIFT 描述子
4 × 4 × 8 = 128 维;4×4 子块,8 方向梯度直方图
SURF 描述子
4 × 4 × 4 = 64 维;4×4 子块,4 方向
PCA-SIFT
20-36 维;用 PCA 降维 SIFT
距离四性质
非负 / 同一(d(x,x)=0)/ 对称(d(x,y)=d(y,x))/ 三角不等式
常用距离
L1(曼哈顿)/ L2(欧氏)/ 余弦 / 马氏距离

陷阱:Harris 角点不抗尺度缩放;SIFT/SURF/MSER 才抗尺度。
