I.考点清单 (Exam Point Checklist)

第十一讲 (L11) 考点 · 基于内容的多媒体检索 (CBIR)

L11-考点 1

语义鸿沟与意图鸿沟

核心
低层视觉特征与高层语义之间的"语义鸿沟 (Semantic Gap)";文本标签难以覆盖图像内容与查询意图的"意图鸿沟 (Intention Gap)"
对比
传统基于文本的检索 vs 基于内容的检索 (CBIR, Query by Example)
易错
混淆"语义鸿沟"和"意图鸿沟"——前者是低层特征↔高层语义,后者是用户查询↔内容描述
频率
中频
L11-考点 2

CBIR 系统框架

核心
离线库图像→特征提取→索引;在线查询图像→特征提取→匹配→结果
关键
三大关键技术:特征提取、特征匹配、高维索引;QBE (Query By Example) 是 CBIR 核心交互模式
易错
忽略"高维索引"环节的重要性——它是支持大规模库检索的工程关键
频率
中频
L11-考点 3

全局特征(颜色/纹理/形状)

核心
颜色直方图、纹理统计量、形状描述子;计算简单但难以应对遮挡、裁剪等攻击,缺乏不变性
弱点
对几何变换(旋转、缩放、裁剪)敏感;丢失空间分布信息
易错
颜色直方图对光照敏感;对"内容相同但颜色不同"的图像区分力差
频率
中频
L11-考点 4

局部特征检测子分类与 Harris 角点

公式
R = det(M) - k·trace(M)²,k = 0.04~0.06;M 是 2×2 图像微分矩阵,由 Ix², IxIy, Iy² 高斯加权
分类
基于特征值 λ₁, λ₂:两者都小→Flat;一大一小→Edge;都大且相近→Corner
分类
检测子分三类:基于角点(Harris/FAST)、基于斑点(SIFT/SURF)、基于区域(MSER)
易错
Harris 不具有尺度不变性;对模糊和噪声敏感;k 是经验常数
频率
高频
L11-考点 5

FAST 角点检测

核心
圆环上若有连续 12 个像素比中心像素更亮(或更暗),则该点为角点
加速
实际使用剪枝先测 1/9/5/13 号像素
优缺
算法简单、速度极快(几毫秒);但角点定义过严、误检率高、不具有尺度不变性
易错
FAST 的"12 点连续"是定义门槛,硬件加速剪枝并非定义
频率
低频
L11-考点 6

尺度空间与 Harris-Laplace

核心
图像多尺度两种方法:图像金字塔(平滑+降采样)与高斯尺度空间(卷积可变尺度高斯核)
关键
特征尺度 = 响应值最大的尺度
流程
Harris-Laplace 生成高斯尺度空间计算角点响应,在尺度空间搜索拉普拉斯响应极大值
易错
高斯尺度空间 ≠ 图像金字塔——前者图片尺寸不变,后者尺寸逐层减半
频率
中频
L11-考点 7

SIFT 检测子(DoG 金字塔)

核心
Lowe 1999/2004;在 DoG (Difference-of-Gaussian) 尺度空间同时检测二维空间与尺度空间的局部极值
步骤
①高斯金字塔→DoG 金字塔;②26 邻域比较极值;③曲线拟合精确定位;④Hessian 删除边缘响应
原理
DoG 是规范化 LoG (Laplacian-of-Gaussian) 的近似,LoG 各向同性具真正尺度不变性
优缺
鲁棒性高(旋转+尺度不变);但实时性不高(1~2 秒)、不具仿射不变性、特征数约 3000
易错
DoG ≈ LoG 但 ≠ LoG;SIFT 不抗仿射;26 邻域是"立方体邻域"(同层 8 + 上下层各 9)
频率
高频
L11-考点 8

SURF 加速鲁棒特征

核心
基于 Hessian 矩阵行列式建立多尺度金字塔;二阶导数近似模板 + 积分图加速
关键
金字塔构建时图片大小不变,只改变高斯核尺度
优缺
速度是 SIFT 的 3~5 倍(描述子环节可达 3~7 倍);但光照鲁棒性略弱于 SIFT,不抗仿射
易错
SURF 金字塔构建方式与 SIFT 不同(图片尺寸不变 vs 逐层降采样)
频率
中频
L11-考点 9

MSER(最大稳定极值区域)

核心
Matas 2002;地形学分水岭概念;按灰度排序(Binsort, O(n)),用 Component Tree 记录连通区域
关键
在宽阈值范围内形状稳定的区域即 MSER
优缺
唯一具有仿射不变性的区域检测子;提取快速、重复率高;但对模糊敏感、信息量少
易错
MSER 是"检测子 + 区域",而 SIFT/SURF 是"检测子 + 描述子"组合
频率
中频
L11-考点 10

SIFT 描述子(128 维向量)

步骤
①像素梯度计算;②局部区块主方向统计;③坐标轴旋转到主方向 + 4×4 分块每块 8 方向 → 4×4×8=128 维;④特征归一化
PCA
PCA-SIFT 在 39×39 邻域内取 3042 维梯度向量,用离线训练得到的 K 维 PCA 投影降至 K 维(20~36)
优缺
128 维区分性强、旋转不变;缺点是维数高、存储匹配代价大
易错
4×4 是分块数,8 是方向 bin 数;PCA-SIFT 是把 3042 维原始梯度降到 K 维,而非 128 维
频率
高频
L11-考点 11

距离度量与性质

公式
d(x,y) ≥ 0, d(x,x)=0, d(x,y)=d(y,x), d(x,k)+d(k,y) ≥ d(x,y)
性质
①非负性 ②同一性 ③对称性 ④三角不等式
常用
欧氏距离(L2)、曼哈顿距离(L1)、余弦距离、马氏距离、汉明距离(二进制描述子)
易错
相似性≠距离,需要从距离转换(如高斯核 e^(-d²/2σ²));不同特征类型选用不同度量
频率
中频
L11-考点 12

群组匹配(空间上下文)

核心
单点特征匹配不稳定,需要群组/几何校验
方法
KNN Vote、RANSAC、WGC (Weak Geometric Consistency)、Bundle of features
易错
RANSAC 适合估计几何变换矩阵;KNN Vote 用最近邻投票做空间投票
频率
低频
L11-考点 13

高维索引(简述)

核心
倒排索引(基于量化词袋)、树型索引(KD-Tree / R-Tree)、哈希索引(LSH 局部敏感哈希)
易错
当维度 > 16 时 KD-Tree 退化为线性扫描,需要近似最近邻 (ANN) 方法
频率
低频(仅作选择题概念)

第十二讲 (L12) 考点 · 深度学习与卷积神经网络

L12-考点 1

深度学习三部曲

核心
①搭建网络结构;②找到合适损失函数;③找到合适优化函数更新参数
关键
损失函数衡量预测与真实吻合度;优化函数通过反向传播更新权重 W
易错
把"损失函数"和"优化函数"混淆——前者衡量吻合度,后者更新参数
频率
中频
L12-考点 2

常用损失函数

分类
交叉熵损失 (Cross Entropy Loss)、Hinge Loss l(y,f(x)) = max(0, 1-y·f(x))(SVM 风格)
回归
均方误差 MSE、L1 损失(平均绝对值误差)
易错
分类≠回归——损失函数选择与任务类型严格对应;Hinge Loss 与 SVM 紧密相关
频率
中频
L12-考点 3

全连接网络处理图像的瓶颈

核心
图像展平后输入全连接网络,权重矩阵参数过多→过拟合
例
1000×1000 RGB 图像 = 3×10⁶ 输入维度,第一层若有 1000 个神经元,仅一层就需要 3×10⁹ 个权重
易错
参数爆炸≠计算慢,关键是过拟合
频率
中频
L12-考点 4

CNN 三大设计思想

局部
每个神经元只感受局部区域(局部感受野),符合图像的空间局部性
共享
同一卷积核在整张图上滑动共享权重,大幅减少参数
多核
使用多个不同卷积核提取不同特征(如边缘、纹理、颜色)
易错
CNN 三大优点——局部关联+参数共享+层级特征表达
频率
高频
L12-考点 5

卷积层关键超参数与输出尺寸

公式
无 padding:output = (N - F) / stride + 1
公式
有 padding:output = (N + 2·padding - F) / stride + 1
参数
(F × F × C + 1) × K(含 1 个 bias),K 为卷积核个数
术语
input / kernel / weights / receptive field / feature map / padding / depth / stride
示例
5×5 输入 + 3×3 卷积核 + stride 1 → 3×3 输出(无 padding: (5-3)/1+1=3)
易错
padding 是单边;公式中 +2·padding 表示左右(或上下)各填充 padding 行/列;输出必须为整数
频率
高频 · 必考计算
L12-考点 6

池化层(Pooling)

作用
保留主要特征的同时减少参数和计算量、防止过拟合、提高泛化能力
类型
Max Pooling(取最大值,更常用)、Average Pooling(取平均值)
位置
通常位于卷积层之间、全连接层之间
易错
池化层没有可学习参数;只是固定运算(与卷积层不同)
频率
中频
L12-考点 7

全连接层(FC)

核心
两层之间所有神经元都有权重连接;通常位于 CNN 尾部
关键
参数量通常最大(VGG 138M 参数大多在 FC 层)
易错
ResNet/GoogleNet 已逐步去掉 FC 层(除输出层外),改用全局平均池化 (GAP)
频率
中频
L12-考点 8

AlexNet(2012 ImageNet 冠军)

历史
top-5 错误率比上一年下降约 10 个百分点,开启深度学习时代
结构
8 层 (C1→C2→C3→C4→C5→F1→F2→F3),5 卷积 + 3 全连接
关键
①百万级 ImageNet 数据;②ReLU 激活函数(解决梯度消失);③Dropout + Data Augmentation;④双 GPU 并行训练
细节
前两次"卷积-ReLU-池化",第三/四次"卷积-ReLU"无池化,第五次"卷积-ReLU-池化",第六/七次"FC-ReLU-Dropout",第八次"FC-Softmax"
易错
ReLU 的关键贡献是"在正区间不会饱和",解决 sigmoid/tanh 的梯度消失
频率
高频
L12-考点 9

VGG(2014)

核心
更深的网络(16-19 层 vs AlexNet 8 层);多个 3×3 小卷积核替代大卷积核
关键
两个 3×3 堆叠 = 一个 5×5 感受野,但参数更少+非线性更多
性能
ILSVRC top-5 错误率 11.7% → 7.3%
参数
138M,主要分布在全连接层
易错
VGG 简单但参数巨大,正是后续 GoogleNet/ResNet 改进的动机
频率
中频
L12-考点 10

GoogleNet / Inception(2014 冠军)

性能
top-5 错误率 11.7% → 6.7%
架构
22 层带参数层(含 pooling 共 27 层),独立成块约 100 个,参数量仅 AlexNet 的 1/12
关键
除输出层外没有 FC 层
演进
Naïve 多核并联;V2 插入 1×1 降维(瓶颈层);V3 用小核替代大核(5×5→两个 3×3)
辅助
辅助分类器解决深度网络的梯度消失问题
易错
1×1 卷积的核心作用是降维/通道变换,不是提取空间特征
频率
高频
L12-考点 11

ResNet 残差网络(2015 冠军)

性能
top-5 错误率 6.7% → 3.57%,深度 152 层
核心
H(x) = F(x) + x,让网络学习残差 F(x) 而非直接学习映射 H(x)
关键
解决深度网络的"退化问题"(不是过拟合,是训练误差随深度增大而增大)
架构
除输出层外没有其他全连接层(用全局平均池化 GAP)
易错
残差网络解决的不是梯度消失,而是深度退化问题;跳连接使得"什么都不做"也容易(学 F(x)=0)
频率
高频
L12-考点 12

CNN 典型网络演进对比

层数
LeNet(5) → AlexNet(8) → VGG(16-19) → GoogleNet(22) → ResNet(152) → Xception(2017)
主线
变深 → 减少参数 → 解决退化
易错
Xception 是 Inception V3 的极端版(深度可分离卷积)
频率
中频
L12-考点 13

数据增强、Dropout、正则化

增强
平移、翻转、对称;随机 crop(256×256 → 224×224);水平翻转;PCA Color Augmentation
Dropout
训练时随机关闭部分神经元(典型 0.5);测试时使用所有神经元;本质是隐式的模型集成
ReLU
f(x) = max(0, x);正区间不会饱和,计算快、收敛远快于 sigmoid;缺点是负区间梯度为 0(Dead ReLU)
易错
Dropout 只在训练时启用,测试时关闭;ReLU 不是零中心化
频率
中频

II.习题 (Exercises)

L11 习题 · 基于内容的多媒体检索

L11-Q1单选 下列哪一项最准确地描述了 CBIR 系统需要克服的"语义鸿沟 (Semantic Gap)"?

  • A. 不同图像采集设备造成的像素差异
  • B. 图像底层视觉特征(如颜色、纹理)与高层人类语义之间的差异
  • C. 用户查询关键词与数据库索引词之间的差异
  • D. 图像压缩前后视觉质量的差异
显示答案与解析
答案 B 解析 语义鸿沟 (Semantic Gap) 指图像底层视觉特征与高层人类理解的概念(如"一只狗在草地上跑")之间的鸿沟;C 选项对应的是"意图鸿沟"。

L11-Q2单选 关于 Harris 角点响应函数 R = det(M) - k·trace(M)² 的描述,错误的是?

  • A. k 是经验常数,取值范围 0.04~0.06
  • B. M 是 2×2 图像微分矩阵,由 Ix², IxIy, Iy² 高斯加权得到
  • C. 当 λ₁, λ₂ 都较大且相近时,R 为较大正值,对应角点
  • D. Harris 检测子对尺度变化具有完全不变性
显示答案与解析
答案 D 解析 Harris 角点检测子不具有尺度不变性,对图像缩放敏感;Harris-Laplace 才具有尺度不变性。A/B/C 都是 Harris 角点的正确描述。

L11-Q3单选 SIFT 检测子采用 DoG (Difference-of-Gaussian) 近似 LoG 的主要原因是?

  • A. DoG 比 LoG 精度更高
  • B. DoG 计算更简单,可减少计算量
  • C. DoG 具有仿射不变性而 LoG 没有
  • D. DoG 在离散像素上比 LoG 更容易实现
显示答案与解析
答案 B 解析 DoG 是规范化 LoG 的近似,两者效果相近但 DoG 通过两个高斯图像相减即可获得,无需计算二阶导数,计算量大幅降低。仿射不变性与 DoG/LoG 选择无关。

L11-Q4多选 下列哪些检测子具有尺度不变性?

  • A. Harris Corner
  • B. SIFT (DoG)
  • C. Harris-Laplacian
  • D. MSER
显示答案与解析
答案 B, C, D 解析 Harris 角点只在单尺度上检测,不具尺度不变性;SIFT 在 DoG 尺度空间检测、Harris-Laplacian 在拉普拉斯尺度空间搜索特征尺度、MSER 在不同灰度阈值范围下检测区域,三者均具有尺度不变性(MSER 还额外具有仿射不变性)。

L11-Q5多选 关于 SIFT 描述子的描述,正确的有?

  • A. 以特征点为中心取 4×4 分块,每块统计 8 个方向的梯度幅值,共 128 维
  • B. 在统计直方图前需要将坐标轴旋转到主方向
  • C. 特征向量最终归一化以去除光照影响
  • D. SIFT 描述子对尺度不变性的实现依赖于分块尺寸的固定
显示答案与解析
答案 A, B, C 解析 D 错——SIFT 的尺度不变性依赖于在特征尺度对应的图像上提取描述子(坐标轴已在 DoG 尺度空间内对齐),而不是分块尺寸固定。

L11-Q6简答 简述 SIFT 检测子的主要步骤,并解释为什么 DoG 可以近似 LoG。

显示答案与解析
答题要点
1. SIFT 步骤:
  • Step 1:构建高斯金字塔(多分辨率金字塔+多层高斯滤波);
  • Step 2:组内逐层相减得到 DoG 金字塔;
  • Step 3:每个像素与 26 邻域(立方体邻域:同层 8 个 + 上下层各 9 个)比较 DoG 响应值找极值;
  • Step 4:对 DoG 函数进行曲线拟合,实现关键点精确定位;
  • Step 5:利用 Hessian 矩阵判断主曲率,删除边缘响应点。
2. DoG 近似 LoG:
  • LoG (Laplacian-of-Gaussian) 算子具有各向同性的斑点响应特性,是真正的尺度归一化算子;
  • DoG 是两个相邻尺度高斯核的差分;
  • 由热扩散方程可得:∂G/∂σ ≈ σ·∇²G,即 LoG 可用相邻尺度高斯差分近似;
  • 因此 DoG 是规范化 LoG 的近似,但计算量仅为 LoG 的一小部分(无需对图像做二阶微分)。

L11-Q8综合应用 设计一个基于内容的商品图像检索系统,要求用户上传一张商品照片,系统在百万级商品库中找到相同或相似的商品。请描述系统的离线建库流程和在线检索流程,并讨论关键技术挑战。

显示答案与解析
答题要点
离线建库流程:
1. 商品库图像预处理(去噪、统一尺寸如 256×256);
2. 特征提取:提取全局特征(颜色直方图、纹理 GLCM)和局部特征(SIFT/SURF 检测子+描述子);
3. 特征聚合:对局部特征采用 Bag-of-Words (BoW) 模型聚类(如 K-means, K=1000)形成视觉词袋;每张图转化为视觉词频直方图;
4. 高维索引:使用倒排索引(基于视觉词)+ 哈希索引 (LSH);
5. 存储特征库与索引。
在线检索流程:
1. 用户上传查询图像,预处理后提取同样的全局+局部特征;
2. 特征量化:将 SIFT 描述子映射到最近的视觉词(NN 搜索);
3. 查询向量构造:直方图表示;
4. 倒排索引查询:返回候选集;
5. 重排序:用空间校验(RANSAC)做几何一致性验证,过滤误匹配;
6. 返回 top-K 最相似商品。
关键技术挑战:
• 语义鸿沟:相同语义商品外观差异大(不同角度/光照/遮挡),需要鲁棒特征;
• 高维索引:SIFT 128 维 + 百万库,精确 KNN 太慢,需要 ANN 近似最近邻;
• 可扩展性:商品类别更新时索引需要增量构建;
• 效率 vs 精度:BoW + 倒排索引牺牲部分精度换取速度。

L11-Q9论述 论述全局特征与局部特征在 CBIR 中的优缺点,并解释为什么现代 CBIR 系统通常采用"局部特征 + 视觉词袋"的组合策略。

显示答案与解析
答题要点
全局特征:颜色直方图、GLCM 纹理、形状描述子等
• 优点:计算简单、维度低、检索速度快;
• 缺点:对遮挡、裁剪、几何变换敏感;丢失空间分布信息;难以应对"语义相似但外观差异"的图像。

2. 通过 BoW 聚类将不定数量的局部特征转化为固定维度的直方图,便于索引和比较;
3. 借鉴文本检索中 TF-IDF + 倒排索引的成熟框架;
4. 局部特征的空间几何信息可通过 RANSAC 等做后验校验,避免完全丢失结构信息。

L12-Q7简答 简述 ReLU 激活函数的优缺点,并解释为什么 AlexNet 使用 ReLU 而不是 sigmoid。

显示答案与解析
答题要点
ReLU 函数:f(x) = max(0, x)
优点:
1. 正区间梯度恒为 1,不存在梯度消失问题;
2. 计算简单(只需比较和取最大),速度远快于 sigmoid 的指数运算;
3. 收敛速度远快于 sigmoid/tanh(实验证明训练速度快约 6 倍);
4. 稀疏激活:负区间输出为 0,使网络具有稀疏性。
缺点:
1. 负区间梯度为 0,可能出现"Dead ReLU"(神经元永远不被激活);
2. 输出不是零中心化,可能影响收敛;
3. 对学习率敏感,过大易导致大量神经元死亡。
AlexNet 选择 ReLU 的原因:
• 解决 sigmoid 在深层网络中的梯度消失问题,使得训练更深的网络成为可能;
• 收敛速度快,可大幅缩短训练时间;
• 计算简单,适合 GPU 并行。

L12-Q8简答 什么是 Dropout?它在训练和测试阶段的行为有什么区别?为什么 Dropout 能防止过拟合?

显示答案与解析
答题要点
Dropout 定义:训练时以概率 p(通常 0.5)随机将神经元输出置零,使其不参与前向传播和反向传播。
训练阶段:每个 mini-batch 随机丢弃部分神经元,相当于在训练不同的子网络。
测试阶段:使用所有神经元,但需要按保留概率 (1-p) 对权重进行缩放(inverted dropout 实现时已在训练时除以 (1-p),测试时无需缩放)。
为什么能防止过拟合:
1. 隐式的模型集成:每次训练相当于训练一个不同的子网络,最终模型相当于大量子网络的集成;
2. 减少神经元共适应:防止某些神经元只在其他特定神经元存在时才有效,迫使每个神经元学习更鲁棒的特征;
3. 类似正则化:等价于在损失函数中加入 L2 正则项;
4. 增加噪声:训练过程的随机性提升泛化能力。
