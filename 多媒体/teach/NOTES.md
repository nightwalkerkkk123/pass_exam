# NOTES — 多媒体系统导论教学工作区

## 教学法（本工作区约定）
- **一节一窄主题**：每个 HTML 微课只解决一个考点或一类计算题；不堆料。
- **每节必含**（AGENTS.md 新增微课规范）：直觉类比 → 关键定义 → 小表格 → 核心内容/例题 → ≥1 道自测题及答案解释 → 下一步建议。
- **计算题给全解**：凡真题命中的计算类主题，必须有一道「按步骤做完」的样例，读者能照着复现。
- **自成一体**：每节独立打开即可读，不依赖对话上下文。

## 视觉规范
- 复用 `exam-review/` 的报纸风设计系统（米色纸 `#f5efe4` + 墨黑 + 酒红 accent `#7c2233`；衬线正文 + 无衬线标签）。
- 全套微课共用同一套 CSS 变量与组件类，保证 21 节看起来出自同一系统。
- 组件：`.intuition`（直觉类比框）、`.def`（定义卡）、`.worked`（例题分步）、`.selftest`（可折叠自测题）。
