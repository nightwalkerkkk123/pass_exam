量化：均匀 / 非均匀 · 标量 / 向量

压缩为什么会"失真"?答案几乎全在这一步——量化。它是有损压缩里信息损失的主要来源,也是老师标红的重点。搞清楚它,JPEG、音频 PCM、率失真理论都能串起来。

直觉类比

想象你要把身高填进表格,但表格只让你选"矮 / 中 / 高"三档。一个 176cm 的人被记成"高",还原时统一说成 180cm——这 4cm 的偏差就永远丢了。量化就是这么回事:把连续(或很多档)的取值,强行塞进少数几个代表值。档位越少,压得越狠,失真也越大。这个"塞"的动作不可逆,所以量化是有损压缩的失真根源。

1.关键定义

量化 (Quantization)
把连续或大集合的输入值,映射到一个小得多的离散输出集合的过程。它是有损压缩方案中"损失(loss)"的主要来源。原始 {0,1,2,…,255} → 缩减为 {16,48,80,…,240} 这类少数代表值。
标量量化 (Scalar Quantization, SQ)
逐个样本独立量化,一次处理一个数值。每个输入被单独映射到一个重建值。
向量量化 (Vector Quantization, VQ)
把若干个连续样本拼成一个 n 维向量,整体一次性量化。用一组 n 维码向量(code vector)作代表,所有码向量构成码本(codebook)。编码端找出与输入向量最接近的码向量,只输出它的索引;解码端用同一份码本按索引取回。SQ 是 VQ 在 n=1 时的特例。
均匀量化 (Uniform Quantization)
把输入域划分成等间隔的区间(bin),间隔宽度处处相等。判决边界 bᵢ 与重建值 yᵢ 都均匀排布。
非均匀量化 (Nonuniform Quantization)
把输入域划分成不等间隔的区间——在信号出现概率高的地方分得更细,把"比特花在最需要的地方"。典型方法:Lloyd-Max 量化、压缩扩展量化(companded)。
量化步长 (Step Size, Δ)
均匀量化中每个区间的宽度,记作 Δ。Δ 越大→档位越少→压缩率越高、失真越大;Δ 越小→越精细、失真越小。

2.两组对照:别把它们搞混

期末选择题最爱在这两组概念上设陷阱。注意:"均匀/非均匀"和"标量/向量"是两个正交的维度——前者说区间怎么分,后者说一次处理几个样本。

均匀 vs 非均匀量化

维度均匀量化非均匀量化
区间间隔等间隔(宽度都是 Δ)不等间隔
核心思想简单,不看数据分布按概率分布分配精度,高概率区更细
适用场景输入近似均匀分布;JPEG 对 DCT 系数的量化输入分布不均(如语音幅度小值居多);Lloyd-Max、μ律/A律压扩
优劣实现最简单,但对非均匀分布不最优同比特下失真更小,但需知道/估计分布

标量 vs 向量量化（真题高频选择题）

维度标量量化 SQ向量量化 VQ
处理单位一次1 个样本(标量)一次n 个样本拼成的向量
代表值单个重建值 yᵢn 维码向量,存于码本
输出每个样本的量化值最近码向量的索引
压缩效果较弱(忽略样本间相关性)更强(利用样本间相关性,同码率下失真更低)
代价 / 场景实现简单、快;PCM、JPEG需训练码本、编码需搜索最近向量,复杂度高;图像/语音编码

一句话记忆:SQ 是"逐个量化",VQ 是"打包量化 + 查码本给编号"。VQ 用一组码向量替代 SQ 的一组标量重建值,靠利用样本间相关性,在相同码率下取得更低失真。

3.为什么要设计量化器?为什么它必然丢信息?

目的:用更少的比特表示信息。原始信号取值多、码字长;量化把取值数量砍到一个小集合,每个值用更短的码字表示,从而压缩数据量。这是有损压缩获得高压缩率的关键一步。

为什么必然损失信息:量化是一个多对一的映射——落在同一区间 [bᵢ₋₁, bᵢ) 内的所有不同输入,都被还原成同一个重建值 yᵢ。既然多个不同输入映射到同一输出,就无法从输出反推出原始输入,信息在这一步被抹平,不可逆。这就是"量化误差"的来源,也是它成为有损压缩失真主要来源的根本原因。

衡量失真常用量化误差 e = x − x̂(重建值与原值之差),以及对全体输入求平均的均方误差 MSE。要算 MSE 需要知道输入的概率分布。

例题 · 均匀量化:求量化输出与量化误差

某 8-bit 灰度值范围 [0, 256),用一个 8 档均匀量化器处理。已知量化步长 Δ = 32,采用中升(midrise)重建:每个区间取其中点作为重建值,即区间序号为 k(k=0,1,…,7)时,重建值 yₖ = (k + 0.5)·Δ。求输入 x = 100 与 x = 200 的量化输出及量化误差。

  1. 先确定区间序号 k = ⌊x / Δ⌋(向下取整)。
    对 x=100:k = ⌊100 / 32⌋ = ⌊3.125⌋ = 3。
    对 x=200:k = ⌊200 / 32⌋ = ⌊6.25⌋ = 6。
  2. 代入重建公式 yₖ = (k + 0.5)·Δ 求量化输出。
    x=100 → y₃ = (3 + 0.5) × 32 = 3.5 × 32 = 112。
    x=200 → y₆ = (6 + 0.5) × 32 = 6.5 × 32 = 208。
    注意这两个输出正是缩减集合 {16,48,80,112,144,176,208,240} 中的元素。
  3. 量化误差 e = x − x̂,取绝对值看偏差大小。
    x=100:e = 100 − 112 = −12,|e| = 12。
    x=200:e = 200 − 208 = −8,|e| = 8。
  4. 验证误差落在合理范围:中升量化器每个区间宽 Δ=32,重建值在中点,最大误差不超过 Δ/2 = 16。两个结果(12、8)都 ≤ 16,合理。
结论:x=100 量化为 112,误差 −12;x=200 量化为 208,误差 −8。若把 Δ 减半到 16(改成 16 档),最大误差降到 8,失真更小但需要更多比特——这就是率失真的权衡。
自测题 1

关于标量量化(SQ)与向量量化(VQ),下列说法正确的是?

  1. SQ 一次处理一个向量,VQ 一次处理一个标量
  2. VQ 把多个样本拼成向量整体量化,输出的是最近码向量在码本中的索引
  3. VQ 是无损的,SQ 才有损
  4. 相同码率下 SQ 的失真通常比 VQ 更低
答案:B

VQ 把 n 个连续样本拼成一个 n 维向量整体量化,编码端找最近的码向量,输出其索引,解码端用同一码本按索引取回。
A 说反了——SQ 处理标量、VQ 处理向量。C 错——量化(无论 SQ/VQ)都是多对一映射,都有损。D 错——VQ 利用样本间相关性,相同码率下失真通常更低,这正是它比 SQ 更强的原因。

自测题 2

用步长 Δ = 16 的均匀中升量化器(重建值 yₖ = (k+0.5)·Δ,k=⌊x/Δ⌋)量化输入 x = 50,量化误差 e = x − x̂ 为?

  1. e = +2
  2. e = −6
  3. e = 0
  4. e = +8
答案:B(e = −6)

k = ⌊50 / 16⌋ = ⌊3.125⌋ = 3;重建值 x̂ = (3 + 0.5) × 16 = 3.5 × 16 = 56;误差 e = 50 − 56 = −6。
解题套路:均匀量化三步走——① k=⌊x/Δ⌋ 定区间;② 代入重建公式求 x̂;③ e = x − x̂。误差绝对值必 ≤ Δ/2 = 8,选项里凡 |e|>8 的直接排除。
