失真度量 MSE/PSNR 与率失真理论

有损压缩必然丢东西——那"丢了多少"怎么量化?丢得越少要花多少码率?这一节把"失真"和"码率"这对冤家钉在一张曲线上。

直觉类比

把一张照片压缩,好比把一段话转述给别人:转述得越简短(码率越低),失真就越大;想一字不差(失真为零),就得把原话完整背下来(码率最高)。率失真理论研究的正是这个权衡:你能容忍多大失真 D,就能算出至少需要多少码率 R。而 MSE/PSNR 则是给"失真到底有多大"打一个客观分数的尺子。

1.关键定义

均方误差 MSE (Mean Square Error, σ²d)
原始信号 x 与重建信号 x̂ 之间误差平方的平均值:MSE = (1/N)·Σ(x − x̂)²。它是失真的能量度量,越小越好、单位是"数值平方",不是分贝。
峰值信噪比 PSNR (Peak Signal to Noise Ratio)
用信号的峰值(8-bit 图像为 255)对失真做归一化,再取对数换成分贝:
PSNR = 10·log₁₀(x²peak / MSE) = 10·log₁₀(255² / MSE)。PSNR 越大越好;它是评价图像/视频质量最常用的客观指标。
率失真函数 R(D) (Rate-Distortion Function)
给定一个可容忍的失真上限 D,R(D) 给出在"失真不超过 D"的前提下,对信源编码所需的最低码率(bit/符号)。它刻画了码率与失真之间理论上无法逾越的权衡边界。

2.三把尺子:SNR vs PSNR vs MSE

考试爱把这三个量摆在一起辨析。核心区别:MSE 是原始误差能量,SNR/PSNR 是取了对数、单位为分贝的比值——区别只在分子用"信号方差"还是"信号峰值"。

指标公式单位/方向
MSE 均方误差(1/N)·Σ(x − x̂)²数值平方;越小越好
SNR 信噪比10·log₁₀(σ²x / MSE)dB;分子是信号方差;越大越好
PSNR 峰值信噪比10·log₁₀(x²peak / MSE)dB;分子是信号峰值²(如 255²);越大越好

易错点:因为对数里的量是"能量比"(平方形式),系数是 10 而不是 20,对数底是 10(常用对数)。只有当公式写成"电压幅值之比"(未平方)时才会出现系数 20,例如 SNR = 20·log₁₀(vsignal/vnoise)——那是同一个式子把平方拿到 log 外面的结果。考场上只要盯住"平方 ↔ 系数 10,幅值 ↔ 系数 20"。

3.率失真理论:码率与失真的权衡边界

率失真理论(Rate-Distortion Theory)给"用多少码率、换多少质量"提供了一个理论框架。它的主角是一条曲线 R(D):

  • 横轴 D:允许的失真上限(如 MSE)。纵轴 R:达到该失真所需的最低码率。
  • 单调下降:允许的失真越大,需要的码率越低——想省码率,就得忍受更多失真。
  • 下凸 (convex):曲线向下弯。越靠近"高质量"一端,每多降一点失真,码率的代价增长得越快(边际成本递增)。
  • D = 0 的端点:失真为零即无损,此时 R(0) 等于信源的熵 H——这是无损压缩码率的理论下界,再也压不动了。
  • D 足够大:失真大到可以"什么都不发",码率降到 0,曲线触及横轴。
简答范式(可背)——"请简述率失真理论/R(D) 曲线的含义"
率失真理论研究码率 R 与失真 D 之间的权衡。率失真函数 R(D) 表示:在失真不超过给定上限 D 的前提下,对信源进行编码所需的最低码率。其 R(D) 曲线具有三个特征:①单调下降——容忍的失真越大,所需码率越低;②下凸——越接近高质量端,降低失真的码率代价越高;③当 D = 0 时对应无损压缩,此时码率等于信源的熵 H。它给出了任何编码方案都无法突破的理论极限,是有损压缩设计的基准。
例题 · 由 MSE 计算 PSNR

一幅 8-bit 灰度图像(像素取值 0–255),经有损压缩后重建,测得均方误差 MSE = 100。求其 PSNR。

  1. 确定峰值:8-bit 图像最大像素值 xpeak = 255,故 x²peak = 255² = 65025。
  2. 代入公式(注意系数是 10、底是 10,不是 20):
    PSNR = 10·log₁₀(255² / MSE) = 10·log₁₀(65025 / 100)。
  3. 算括号内比值:65025 / 100 = 650.25。
  4. 取常用对数:log₁₀(650.25) ≈ 2.813。
  5. 乘系数 10:PSNR ≈ 10 × 2.813 ≈ 28.1 dB。
结果:PSNR ≈ 28 dB。记住这个"锚点":MSE = 100 → PSNR ≈ 28 dB。若误用系数 20,会算出约 56 dB,整整差一倍——这正是阅卷常见扣分点。

4.为什么这节重要

失真度量和率失真理论是整个有损压缩(JPEG、MPEG、量化)的评价基石:没有 PSNR 就无法客观比较两种压缩算法孰优孰劣;没有 R(D) 就说不清"多花一倍码率究竟买回多少画质"。真题里,PSNR 常作为计算题的一小问,而率失真理论几乎是简答题的钉子户——本节的可背范式请务必吃透。下一节的"量化"正是产生失真的源头。

自测题 1

某 8-bit 图像压缩后 MSE = 255²/100 ≈ 650(即比值 255²/MSE = 100)。其 PSNR 约为多少?

  1. 10 dB
  2. 20 dB
  3. 40 dB
  4. 100 dB
答案:B 20 dB

题目已给 255²/MSE = 100。直接代入 PSNR = 10·log₁₀(100) = 10 × 2 = 20 dB。
陷阱:若错用系数 20,会得 40 dB(选项 C)——这就是"系数是 10 不是 20"的考点。凡对数里是"能量/平方比",系数一律取 10。

自测题 2

关于率失真函数 R(D),下列说法错误的是?

  1. R(D) 随 D 增大而单调下降
  2. D = 0 时 R(D) 等于信源的熵,对应无损压缩
  3. R(D) 曲线是下凸的
  4. D 越小所需码率越低,故高质量图像最省码率
答案:D

D 越小(要求越高质量),所需码率 R 越高而非越低——高质量恰恰最费码率,D 说反了。A/B/C 都是 R(D) 曲线的正确特征:单调下降、D=0 处等于熵、下凸。
记忆钩:"想清晰就掏钱"——失真越小(越清晰),码率账单越贵。
