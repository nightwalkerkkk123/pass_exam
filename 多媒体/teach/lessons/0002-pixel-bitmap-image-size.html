像素 · 位图 · 图像数据量计算

一张数字图像,拆到底就是一张数字表格。搞清楚"一个格子占几个字节",这门课第一类计算题——图像未压缩存储量——就再也不会失分。

直觉类比

把一张图像想成方格纸:每个小格子就是一个像素 (pixel),格子里填的数字就是这个点的颜色。方格纸有多少行多少列,就是图像分辨率(如 512×512)。整张填满数字的方格纸,就是位图 (bitmap)——一个二维数组。而"这张图占多少空间",本质就是:格子数 × 每个格子记几个字节。全部计算题都从这一句展开。

1.核心定义:从像素到位图

像素 (Pixel)
图像的最小单位,一个点。它记录该位置的颜色/亮度值。灰度图里一个像素是 0~255 的一个数;彩色图里是 (R, G, B) 三个数。
图像分辨率 (Image Resolution)
图像在水平和垂直方向上的像素数,写作 宽 × 高(如 512 × 512、640 × 480)。总像素数 = 宽 × 高。分辨率越高,像素越多,数据量越大。
位图 (Bitmap)
表示图像的二维数组——每个数组元素就是一个像素值。"表示图像或图形的二维数组就称为位图。"
位平面 (Bitplane)
把一张 8-bit 图像看成 8 张 1-bit 图像的叠加:每个像素的第 k 位单独抽出来拼成一张平面,就是第 k 个位平面。高位平面决定图像结构、形状与明暗;低位平面多为噪声与细纹。用于压缩、加密隐写、去噪。
24-bit 彩色图 (24-bit Color Image)
RGB 模型下 R(8-bit) + G(8-bit) + B(8-bit),每个像素占 3 个字节。可表示 256×256×256 ≈ 1680 万种颜色,又称真彩色。有些系统存成 32-bit(多 1 字节存特效/透明信息)。

2.每种图像:一个像素占几个字节

数据量计算的全部秘密,就藏在下面这张"每像素字节数"表里。记住它,套公式即可:

图像类型每像素位数每像素字节数数据量公式(字节)
二值图像 (Binary / 1-bit)1 bit1/8 B宽 × 高 ÷ 8
8-bit 灰度图8 bit1 B宽 × 高 × 1
24-bit 彩色图24 bit3 B宽 × 高 × 3
32-bit 彩色图32 bit4 B宽 × 高 × 4

由此立刻得到一个常考结论:同分辨率下,8-bit 灰度图的数据量是 24-bit 彩色图的 1/3(每像素 1 B vs 3 B)。这正是讲义思考题的答案。

换算关系务必背牢:1 Byte = 8 bit、1 KB = 1024 B、1 MB = 1024 KB。

3.bit 与 Byte:计算题最大的坑

题目常故意问"多少比特 (bit, b)"或"多少字节 (Byte, B)",两者差 8 倍。看讲义给的 640×480 标准例子,同一张图两种口径:

图像按 bit 算按 Byte 算
640×480 · 8-bit 灰度640×480×8 = 2,457,600 b = 2400 Kb640×480×1 = 307,200 B = 300 KB
640×480 · 24-bit 彩色640×480×24 = 7,372,800 b = 7200 Kb640×480×3 = 921,600 B = 900 KB

避坑口诀:先看单位是小 b 还是大 B。乘位数(×8 / ×24)得到的是 bit;乘字节数(×1 / ×3)得到的是 Byte。答错单位,整道计算题清零。

例题 · 512×512 的 24-bit 彩色图未压缩数据量

求一张分辨率为 512 × 512 的 24-bit 彩色图像,未压缩时占多少字节?换算成 KB 是多少?

  1. 算总像素数:512 × 512 = 262144 个像素。
  2. 确定每像素字节数:24-bit 彩色 = R+G+B = 3 个字节 / 像素。
  3. 套数据量公式:512 × 512 × 3 = 786432 字节 (B)。
  4. 换算成 KB(除以 1024):786432 ÷ 1024 = 768 KB。
  5. 再换算成 MB(再除以 1024):768 ÷ 1024 = 0.75 MB。
结果:786432 B = 768 KB = 0.75 MB。与讲义"512×512 24-bit colour image = 0.75 MB"完全一致。
若题目问的是 bit:512 × 512 × 24 = 6,291,456 b,正好是字节数的 8 倍——别把单位搞反。
自测题

一张分辨率为 640 × 480 的 24-bit 彩色图像,未压缩时的数据量是多少字节?

  1. 307,200 B(≈ 300 KB)
  2. 921,600 B(≈ 900 KB)
  3. 7,372,800 B(≈ 7200 KB)
  4. 2,457,600 B(≈ 2400 KB)
答案:B 921,600 B(≈ 900 KB)

24-bit 彩色每像素 3 字节,所以 640 × 480 × 3 = 921,600 B = 900 KB。
为什么其它是坑: A(307,200)是把它当 8-bit 灰度(×1)算的;C(7,372,800)是按 bit 算的(×24),单位搞反了;D 是灰度图按 bit 算(×8)。解题套路:先认单位(B 还是 b),彩色乘 3、灰度乘 1;要 bit 就乘位数(24 或 8)。
