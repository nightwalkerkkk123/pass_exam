# 《啊哈！算法》提取说明

- 原始文件：`technical_books · GitHub.pdf`（文件名具有误导性，实际是《啊哈！算法》一书的 PDF，著者「啊哈磊」，人民邮电出版社·图灵原创，2014年6月第1版，ISBN 978-7-115-35459-4）
- 全书 258 页（PDF 页码，0-indexed）：
  - PDF 页 10–67：有完整可提取文字层（pypdf 直接提取），对应第1章「排序」、第2章「栈、队列、链表」
  - PDF 页 0–9、68–257：**无文字层，纯图片扫描**，已用 Tesseract OCR（chi_sim+eng，300dpi）识别
- OCR 覆盖第3~9章（枚举、万能的搜索、图的遍历、最短路径、神奇的树、更多精彩算法、微软亚洲研究院面试）。这些章节含大量手绘漫画插图与截图代码，OCR 质量参差：部分页面文字清晰可读，部分页面（尤其是插图密集页）识别为乱码。
- `啊哈算法-提取文本.txt` 中每页前有 `----- PDF_PAGE N (pypdf|ocr) -----` 标记，标明该页的提取方式，供后续核实溯源。
- 生成的 skill（`skill/aha-algorithms/`）中，凡是依赖 OCR 无法识别、改用标准算法讲解补充的内容，均在正文中明确标注，不冒充是原书文字的忠实复现。
