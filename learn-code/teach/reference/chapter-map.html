几个贯穿全书的关键头文件

文件装的是什么
kernel/proc.h进程控制块 struct proc、CPU 状态——读任何子系统前都会回头看它
kernel/defs.h内核内部所有函数的声明总表(相当于内核的"目录")
kernel/param.h各种上限常量(进程数、打开文件数、CPU 数……)
user/user.h用户程序能调用的系统调用 + 库函数声明
