⚠️ 易错雷区清单：
  • 银行家 ≠ 死锁检测：银行家是「避免」（事前试探安全序列），资源图化简是「检测」（事后判断），别混。见 0058 / 0003 / 0046。
  • 页面置换算法要写清缺页判定：命中不算缺页，Belady 异常只出现在 FIFO。见 0061 / 0070。
  • 磁盘调度别忘起始磁头位置和方向：SCAN/CSCAN 要不要到达边界，直接影响总道数。见 0062 / 0007。
  • 最大文件容量单位换算：盘块大小与地址项长度先统一单位，别在 KB/B 上翻车。见 0009。
  • PV 默写顺序：先判断信号量含义再写 P/V，互斥信号量初值为 1、资源信号量初值为个数。见 0050；读者写者两把锁别混、if(readcount==0) 不要写 1，见 0051；哲学家奇偶分拿别把 if/else 写成一模一样，见 0052。期末 6 道代码题（PQR+BufI/O / 前趋图 / 苹果桔子 / 哲学家 / 读者写者 / 司机售票员）一锅端见 0080。
  • 分页/分段必查段长：分段必须检查段内地址 < 段长，否则越界；分页天然不会越界。见 0060。
  • 死锁四条件必须同时满足：不是满足一个就死锁，四个同时满足才会死锁；破坏任一条件可预防。见 0068。
