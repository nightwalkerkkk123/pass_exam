分页存储管理 · 程序的"格子"和内存的"格子"

目标：15 分钟装下"页面=物理块=4KB"+"页表是啥"+"逻辑地址怎么变物理地址"——下节课页面置换就在这之上操作

1. 一句话 + 酒店类比

把程序和内存都切成同样大小的"小格子"（4KB 一格），用一张"对应单"（页表）记下程序第几格住进了内存第几格。
📋 旅客行程单（= 整个程序）
第 0 项：8:00 早餐
第 1 项：9:30 开会
第 2 项：14:00 游泳
第 3 项：18:00 晚饭
第 4 项：20:00 看电影
按"半天一个活动"切
🏨 酒店房间（= 整个内存）
305 房（空闲）
312 房（空闲）
408 房（空闲）
510 房（空闲）
607 房（空闲）
每个房间一样大（标准间）

↓ 旅客订房单（= 页表）：活动 0 → 305，活动 1 → 510，活动 2 → 312，活动 3 → 408，活动 4 → 607

记住 3 件事：
1. 行程单每"半天"切一刀 = 程序的"页面"（大小固定 4KB）
2. 酒店每个房间一样大 = 内存的"物理块"（大小固定 4KB，跟页面必须一样大）
3. 订房单 = 页表，OS 替我们维护，告诉我们"活动 N 在房间 X"

2. 关键术语 2 列对照表（4 行，背）

程序侧（虚拟 / 程序员看到）内存侧（物理 / 实际存在）
页面（Page）：程序切出的"小格子"，大小 4KB 物理块 / 页框（Frame）：内存切出的"小格子"，大小 4KB
页号：第几个页面，编号 0, 1, 2, ... 物理块号：第几个物理块，编号 0, 1, 2, ...
逻辑地址 = 页号 + 页内偏移（程序员写的） 物理地址 = 物理块号 + 页内偏移（实际送进内存的）
页表 = 左侧"页号" → 右侧"物理块号"的对应单，OS 维护

3. 逻辑地址长什么样？

程序员写一个地址（比如 0x1234），OS 把它拆成两段：

页号 P
页内偏移 W
逻辑地址 = 页号 × 页大小 + 页内偏移

CPU 想用逻辑地址 P, W 时，去查页表，假设查到 P → F，那实际访问的物理地址是：

物理块号 F
页内偏移 W
物理地址 = F × 页大小 + W

⚠️ 页内偏移 W 在变换前后一字不变（只换块号，不换位置）

4. 一个例子把全过程走一遍

已知：页大小 = 1KB（1024 字节），页表 = [0→2, 1→5, 2→0]，逻辑地址 = 2500。

第 1 步：把逻辑地址拆成"页号 + 页内偏移"

页号 P  = 2500 / 1024 = 2        （整除）
页内偏移 W = 2500 % 1024 = 452   （取余）

第 2 步：查页表，找物理块号

📋 页表（页号 → 物理块号）
页号物理块号
02
15
20

页号 2 → 物理块号 0

第 3 步：拼物理地址

物理地址 = 物理块号 × 页大小 + 页内偏移
        = 0 × 1024 + 452
        = 452
答：逻辑地址 2500 对应的物理地址是 452。
易错点：页内偏移 452 一定要保留到物理地址里——只换"块号"，不换"块内位置"。

5. 必背 3 句话（考试原话）

① 页面大小 = 物理块大小（必相等，否则对不上）。
② 逻辑地址 = 页号 + 页内偏移；物理地址 = 物理块号 + 页内偏移。
③ 物理地址 = 物理块号 × 页大小 + 页内偏移（整除取号，取余取位置）。

6. 自检（先做再翻答案）

Q1（口述） 用一句话告诉我：什么是"分页"？
（提示：程序和内存两边都切了 + 一张对应单）
Q2（口述） 页面大小是 4KB，物理块大小可以是 8KB 吗？为什么？
Q3（计算） 页大小 = 1KB，逻辑地址 = 1500，求页号和页内偏移。
Q4（选择） 页大小 1KB，页表 = [0→1, 1→2, 2→3]，逻辑地址 1500 对应的物理地址是？
A. 1500（不变）
B. 476（= 0×1024 + 476）
C. 2524（= 2×1024 + 476）
D. 1500 + 1 = 1501
Q5（计算） 页大小 = 4KB，逻辑地址 = 5000，求页号和页内偏移。（页大小变了，注意！）
答案对照
A1：把程序和内存都切成同样大小的小格子（4KB），用页表记下程序第几格对应内存第几格。
A2：不行。页面和物理块必须等大，否则"格子"对不上号（装不进去 / 装不满）。
A3：P = 1500 / 1024 = 1，W = 1500 % 1024 = 476。
A4：C。P = 1500 / 1024 = 1，W = 1500 % 1024 = 476，查页表 F = 页表[1] = 2，物理地址 = 2 × 1024 + 476 = 2524。
A. 错：忘了查页表，块号默认当成 0；B. 错：把 F 当成 0（漏查页表）；C. 对；D. 错：无关的"加 1"干扰项。
A5：P = 5000 / 4096 = 1，W = 5000 % 4096 = 904。
老师改卷最爱挑的 2 个坑：
1. 页内偏移不参与变换——从逻辑地址抄到物理地址，一字不变（自己算一遍才能保证不错）。
2. 页大小单位——4KB = 4096 字节，不是 4000；1KB = 1024 字节，不是 1000。算之前先看清单位。

7. 完整流程速记（手算时按这 3 步走）

已知：页大小 L，页表，逻辑地址 A

① 拆地址   P = A / L           （整除 → 页号）
          W = A % L           （取余 → 页内偏移）

② 查页表   F = 页表[P]         （页号 → 物理块号）

③ 拼地址   物理地址 = F × L + W
