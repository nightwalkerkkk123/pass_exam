# 计算机操作系统（汤小丹第四版）

## 核心知识体系
### 操作系统基础层
- **核心定义**：操作系统是控制和管理计算机硬件和软件资源、合理调度各类作业、方便用户使用的程序集合，是计算机系统最核心的系统软件。
- **四大本质特征**：
  1. **并发性（Concurrence）**：多个事件在同一时间间隔内发生，是OS最重要的特征
  2. **共享性（Sharing）**：系统资源可供多个并发进程共同使用
  3. **虚拟性（Virtual）**：通过时分/空分复用技术将物理实体抽象为多个逻辑实体
  4. **异步性（Asynchronism）**：进程以不可预知的速度向前推进，但运行结果可重复
- **五大基本功能**：处理机管理、存储器管理、设备管理、文件管理、用户接口
