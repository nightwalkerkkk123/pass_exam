题目是**文件存储空间管理中的成组链接法**。

题目大意是：

> **文件管理作业：**
> 设空闲存储空间管理采用**成组链接法**，每 **5 块**为一组。当前空闲盘块号栈的内容从 **S.free** 开始依次为：
>
> ```text
> 3，1，2，3
> ```
>
> 试写出以下操作后的**空闲盘块号栈和空闲盘块总数的变化过程**：
>
> ① 回收第 4 块；
> ② 再回收第 7、8 块；
> ③ 分配 1 块；
> ④ 分配 3 块。

其中开头的 `3` 表示当前栈里有 **3 个空闲盘块**，后面的 `1，2，3` 才是盘块号。

从投影内容可读到：

```text
每 5 块为一组；
当前空闲盘块号栈的内容从 S.free 开始依次为：
3，1，2，3
```

这里第一个 `3` 不是盘块号，而是**栈中空闲盘块数**。

所以初始状态是：

```text
S.free = [3, 1, 2, 3]
```

含义是：

```text
当前栈中有 3 个空闲盘块：1、2、3
```

假设栈顶在右边，则当前栈顶是盘块 `3`。

------

# 一、成组链接法规则

每组最多 5 块。

## 1. 回收空闲盘块

如果当前栈未满：

```text
直接把回收的盘块号压入栈
空闲块数 +1
```

如果当前栈已满：

```text
把当前栈内容写入新回收的盘块中
然后让这个新回收的盘块号成为新栈中的唯一盘块
```

也就是：

```text
原来的栈作为“下一组信息”存入新盘块
新盘块成为当前组的代表
```

------

## 2. 分配空闲盘块

如果当前栈中空闲块数大于 1：

```text
直接弹出栈顶盘块
空闲块数 -1
```

如果当前栈中空闲块数等于 1：

```text
说明这个盘块中保存着下一组空闲盘块信息
先把该盘块中的内容读入 S.free
再把该盘块分配出去
```

------

# 二、初始状态

```text
S.free = [3, 1, 2, 3]
```

空闲盘块为：

```text
1，2，3
```

空闲盘块总数：

```text
3
```

------

# 三、① 回收第 4 块

当前栈中有 3 块，还没满，因为每组最多 5 块。

所以直接压入 4。

```text
回收前：
S.free = [3, 1, 2, 3]

回收第 4 块后：
S.free = [4, 1, 2, 3, 4]
```

空闲盘块总数：

```text
3 → 4
```

------

# 四、② 再回收第 7、8 块

## 先回收第 7 块

当前：

```text
S.free = [4, 1, 2, 3, 4]
```

栈中有 4 块，还没满，所以直接压入 7。

```text
回收第 7 块后：
S.free = [5, 1, 2, 3, 4, 7]
```

空闲盘块总数：

```text
4 → 5
```

此时栈满了。

------

## 再回收第 8 块

当前栈已经满：

```text
S.free = [5, 1, 2, 3, 4, 7]
```

按照成组链接法：

```text
把当前 S.free 的内容写入第 8 块中
然后 S.free 中只保存第 8 块
```

也就是说，第 8 块里面保存：

```text
[5, 1, 2, 3, 4, 7]
```

新的超级块空闲栈变为：

```text
S.free = [1, 8]
```

空闲盘块总数：

```text
5 → 6
```

注意：此时看起来 S.free 里只有 `8`，但 `8` 里面保存着下一组空闲盘块 `1、2、3、4、7`。

------

# 五、③ 分配 1 块

当前：

```text
S.free = [1, 8]
```

说明当前栈中只有 1 个空闲盘块 `8`。

按照成组链接法：

```text
先把第 8 块中保存的下一组信息读入 S.free
再把第 8 块分配出去
```

第 8 块中保存的是：

```text
[5, 1, 2, 3, 4, 7]
```

所以分配第 8 块后：

```text
分配出去的盘块：8

S.free = [5, 1, 2, 3, 4, 7]
```

空闲盘块总数：

```text
6 → 5
```

------

# 六、④ 分配 3 块

当前：

```text
S.free = [5, 1, 2, 3, 4, 7]
```

栈顶在右边，所以依次分配：

```text
7，4，3
```

------

## 分配第 1 块

分配栈顶 `7`。

```text
分配前：
S.free = [5, 1, 2, 3, 4, 7]

分配 7 后：
S.free = [4, 1, 2, 3, 4]
```

空闲盘块总数：

```text
5 → 4
```

------

## 分配第 2 块

分配栈顶 `4`。

```text
分配前：
S.free = [4, 1, 2, 3, 4]

分配 4 后：
S.free = [3, 1, 2, 3]
```

空闲盘块总数：

```text
4 → 3
```

------

## 分配第 3 块

分配栈顶 `3`。

```text
分配前：
S.free = [3, 1, 2, 3]

分配 3 后：
S.free = [2, 1, 2]
```

空闲盘块总数：

```text
3 → 2
```

------

# 七、完整过程汇总表

| 操作               | 分配/回收的盘块 | S.free 变化                                    | 空闲盘块总数 |
| ------------------ | --------------- | ---------------------------------------------- | ------------ |
| 初始               | —               | `[3, 1, 2, 3]`                                 | 3            |
| 回收第 4 块        | 回收 4          | `[4, 1, 2, 3, 4]`                              | 4            |
| 回收第 7 块        | 回收 7          | `[5, 1, 2, 3, 4, 7]`                           | 5            |
| 回收第 8 块        | 回收 8          | 第 8 块保存 `[5,1,2,3,4,7]`，`S.free = [1, 8]` | 6            |
| 分配 1 块          | 分配 8          | 读出第 8 块内容，`S.free = [5, 1, 2, 3, 4, 7]` | 5            |
| 分配 3 块，第 1 块 | 分配 7          | `[4, 1, 2, 3, 4]`                              | 4            |
| 分配 3 块，第 2 块 | 分配 4          | `[3, 1, 2, 3]`                                 | 3            |
| 分配 3 块，第 3 块 | 分配 3          | `[2, 1, 2]`                                    | 2            |

------

# 八、最后答案

最终：

```text
最终 S.free = [2, 1, 2]
```

最终空闲盘块为：

```text
1，2
```

最终空闲盘块总数：

```text
2
```

整个过程中实际分配出去的盘块依次是：

```text
8，7，4，3
```

回收的盘块是：

```text
4，7，8
```

注意：如果你老师规定“栈顶在左边”，分配顺序会反过来；但多数教材和成组链接法题目默认第一个数字是数量，后面的盘块号中**最后一个是栈顶**。
