💡 使用方法

建议 1：先 lesson 后题库
lesson 是「讲故事」让你懂，题库是「让你写出来」证明你懂。
顺序：lesson → 题库对应章节 → 错题回 lesson。
建议 2：题库要做 2 遍
第 1 遍：开卷做，看每题的 explanation 字段（解析）。
第 2 遍：闭卷做，能答对才算真的会。
建议 3：错题本
所有答错的题，抄题号 + 错因到错题本。
仿真期末卷前 1 天只看错题本，效率最高。
建议 4：注意老师课件的「作业题」
本套题库的简答/计算题有 ★ 标记的是直接来源于老师课件的作业题/思考题，一定要会做。

🛠️ 题库使用方式

每个题库 HTML 都是可交互的：

  • 选择题：点选项
  • 判断题：点「正确」或「错误」
  • 填空题：在框里输入答案（不区分大小写、忽略标点）
  • 简答/计算：参考题，参考答案在「提交批改」后显示
  • 「提交批改」按钮：自动判分，显示每题对错和解析
