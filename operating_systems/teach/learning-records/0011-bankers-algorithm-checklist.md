## B. 四个量(等 A 懂了再引入,一次一个)
- [ ] B1 Max = 客户声明的最高额度
- [ ] B2 Allocation = 已经借走的
- [ ] B3 Need = 还差多少才借满 = Max − Allocation
- [ ] B4 Available = 金库还剩多少

## C. 安全性算法(滚雪球)
- [ ] C1 为什么先满足"还需最少、现在就够给"的人
- [ ] C2 一个人跑完后,金库为什么变多(他把全部还回来)
- [ ] C3 能一路滚到所有人跑完 = 安全;中途卡住 = 不安全
- [ ] C4 能自己手算单一资源的例题
- [ ] C5 多类资源:数字变向量,逐列比较/相加

## D. 完整银行家算法
- [ ] D1 收到请求先查:Request ≤ Need 且 ≤ Available
- [ ] D2 试探分配 → 跑安全性检查 → 安全才真给,不安全撤销让它等
- [ ] D3 "避免" vs "预防"的区别(考试常问)
