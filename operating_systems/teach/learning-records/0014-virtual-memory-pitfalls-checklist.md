## A. 局部性 + 请求分页(第一拍,已口头验证)
- [x] A1 局部性原理:一段时间只用一小撮页 → 不必全装入(用户独立答对)
- [ ] A2 虚拟存储器 = 靠局部性,用请求分页按需调页
- [ ] A3 请求分页页表项:状态位P/访问字段A/修改位M/外存地址(填空高频)
- [ ] A4 缺页代价大:EAT=(1-p)·访存+p·缺页处理,读磁盘慢几万倍

## B. Belady 异常(第二拍)
- [ ] B1 现象:FIFO 下增加页框,缺页率反而升高
- [ ] B2 **只有 FIFO 会;LRU/OPT(栈式算法)不会** ← 常考对比
- [ ] B3 教材例:1 2 3 4 1 2 5 1 2 3 4 5 → 3框9次 vs 4框10次

## C. 抖动(第三拍)
- [ ] C1 定义(知识清单原话):页面频繁换入换出,系统效率急剧下降
- [ ] C2 根因:分配页框太少,装不下常用页,反复换
- [ ] C3 工作集=一段时间真正在用的页集合(老师未重点讲,了解)
