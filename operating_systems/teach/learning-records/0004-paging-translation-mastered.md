# L3 通过:掌握分页地址变换(逻辑地址→物理地址)

学习者通过 L3 双通关:① 复述"页号会变(换成物理块号)、页内偏移不变"——抓住了分页最反直觉的核心;② 准确填出 `物理地址 = 物理块号 × 页大小 + 页内偏移`。

**Evidence**：L3 三步法理解到位,通关复述正确。
**Implications**：
- 分页地基已立,可以进 **页面置换(Ch5)**。
- 页面置换需要先点明"虚拟存储/缺页"的前提(为什么页会不在内存),用 1-2 句桥接即可,不必单开一课。
- 页面置换拆两课(同磁盘调度套路):**L4 = 缺页概念 + 画表法 + FIFO**(最简单);**L5 = LRU + OPT + Belady 异常**。
- "酒店房间"比喻用于分页;页面置换换新比喻避免串味。
