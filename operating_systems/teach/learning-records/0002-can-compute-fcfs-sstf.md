# L1 通过：能正确计算 FCFS / SSTF 总寻道距离（但需盯住"逐段相加"的措辞）

学习者独立算对热身题(磁头40,请求50,20,60,FCFS=80),证明她**实际会逐段累加移动量**——这是新地板:磁盘调度的"算距离"机制已掌握,可以往 SCAN/CSCAN 上叠。

**Evidence**：口算 40→50→20→60 = 10+30+40 = 80,正确。

**需盯的隐患(高价值,预测未来坑)**：她复述概念时说成「起点磁头到**终点**磁头号的差值绝对值」——字面是 `|起点−终点|`,这在磁头折返时会算错(她那题 |40−60|=20≠80)。已当场纠正为「**每一段** |从−到| 全加」。
**Implications**：教 SCAN/CSCAN 时要特别强化"折返/跳回的每一段都要数",尤其 CSCAN 跳回那一大段最容易被她按"起点终点"思路漏掉。已记入 [[GLOSSARY.md]]。
