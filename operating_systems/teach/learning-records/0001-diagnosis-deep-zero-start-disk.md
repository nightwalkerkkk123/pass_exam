# 诊断（2026-06-24）：四大计算题均为"深的零"，分页也已全忘 → 第一课从磁盘调度切入

学习者自评：PV / 页面置换 / 磁盘调度 / 混合索引 **四个计算大题"连概念都不清楚"**，且"分页完全不记得了"。这把起点钉死在最底层——不能假设任何存储/进程前置知识。

**Implications**：
- 不能按教材章节顺序（Ch2→…）教，要按**前置依赖最少 + 最快拿到一次胜利**排序。
- 选定**磁盘调度**作为第一个计算题：零前置（不需要进程/分页/文件系统），本质只是"磁头在数轴上移动，把移动距离加起来"，深的零起点 15 分钟可做对。
- 页面置换需先补"分页/缺页"（她已全忘），是两步工程，往后排。
- PV 最抽象，往后排。
- 第一课只教"如何算磁头总移动距离"，用 FCFS + SSTF 两条最简单规则演示；SCAN/CSCAN 的"方向+折返"陷阱留第二课。
- 通过条件沿用工作区惯例：她能用自己的话复述"总移动距离怎么算"。
