一、死锁：四条件 ↔ 破坏手段（最易混，重点）

必要条件含义预防＝破坏它的做法
互斥资源同时只能一个进程用一般不破坏（资源天生独占）
请求和保持占着旧的，又要新的，且不放手一次性把资源全部分配
不剥夺已占有的不被强行夺走申请不到，就释放已占有的
环路等待形成 A等B→B等C→…→绕回A 的圈资源按编号递增顺序申请
死记口诀： 全分配→破请求保持；不到就放→破不剥夺；按序申请→破环路等待。
另外：银行家算法 = 避免（不破坏任何条件，靠预演绕开不安全状态），不是预防！

二、请求分页：页表项四个字段（第 10 题考点）

字段管什么别混
状态位 P该页在不在内存（P=0 → 缺页）管“在不在”，不是“改没改”
访问字段 A记录最近访问情况/次数，供置换算法（LRU）参考管“用得多不多”
修改位 M该页改过没有；改过→换出要写回外存，没改→直接丢管“要不要写回”
外存地址该页在磁盘的位置，缺页时按它去取
一句话区分 P / A / M： P = 在不在，A = 用没用，M = 改没改。

三、各种算法缩写（别张冠李戴）

页面置换算法

OPT最佳置换：淘汰“以后最久不用”的页。无法实现，只作评价标准。
FIFO先进先出：淘汰最早进入内存的页。唯一会 Belady 异常的
LRU最近最久未使用：淘汰最久没被访问的页。看“访问时间”
LFU / NRU(Clock)LFU=最少使用（看次数）；Clock/NRU=最近未用（改进型用 A、M 位）
最爱考的混淆：FIFO 看“进入时间”，LRU 看“访问时间”。

磁盘调度算法

FCFS先来先服务：按请求到达顺序
SSTF最短寻道时间优先：每次选离磁头最近的
SCAN扫描 / 电梯算法：单向走到头再返回
CSCAN循环扫描：到头后直接跳回起点，单向服务

四、I/O 与设备：缩写

DMA直接存储器访问（Direct Memory Access）：数据成块直达内存，只在块首尾需 CPU
通道I/O 处理机：CPU 发一条指令，通道自己跑“通道程序”，管一组块
SPOOLing假脱机：把独占设备（打印机）→ 可共享的虚拟设备（磁盘输出井排队）
缓冲缓和 CPU 与 I/O 速度不匹配；单/双/循环缓冲、缓冲池
TLB / 快表放在高速缓存里的部分页表，加速地址变换（少访存一次）

五、进程 / 同步 / 银行家：术语

PCB进程控制块：记录进程信息，是进程存在的唯一标志
TCB / LWPTCB=线程控制块；LWP=轻型进程（内核控制线程）
PVP=wait（申请，S−1）；V=signal（释放，S+1）。信号量 S>0=可用资源数；S<0 时 |S|=等待进程数
EAT有效访问时间 = (1−缺页率)×访存 + 缺页率×缺页处理
Max/Alloc/Need银行家：Need = Max − Allocation；Available=系统剩余；安全性算法里 Work 初值 = Available

📌 还想多记一点（点开）：容易忽略的几个
  • 独占/共享/虚拟设备：打印机=独占；磁盘=共享；SPOOLing 造出的=虚拟。
